NYTimes APIs

The NYTimes Article Search API is used to look up articles by keyword. This can be used with different parameters listed in the API documentation, which can be found here: NYTimes APIs. This Article Search API allows the GET method to communicate and returns the JSON object as an output. This API's primary purpose is to build a basic demo application to search for articles based on specific keywords. This can help in tracking the recent and old news based on a certain timeline.
